AWS-hosted Crowdsourcing Platform in React.js

The Goals

Design and develop a crowdsourcing platform for entrepreneurs to raise independent funds for any # of project. The website is built with ReactJS and Amazon Web Services: hosted with Amazon S3, process requests with Amazon API Gateway and Lambda, manage data with AWS RDS and MySQL.

What I did

  • Created frontend interface using React framework and designed user interface (UI) using CSS.
